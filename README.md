# Excel Automation Project

## 📌 Overview
This project automates Excel data processing using Python.

## 🧠 Features
- Reads Excel file
- Calculates total sales
- Computes percentage contribution
- Generates a new report automatically

## 🛠️ Technologies Used
- Python
- pandas
- openpyxl

## ▶️ How to Run
1. Install dependencies:
   pip install pandas openpyxl

2. Run the script:
   python main.py
